The Ashlar Protocol · Milestone 0 gate

Does the stone feel like it’s on your side?

A framing pre-test. Before we build the visual system, we need to know whether “you are a stone being worked” lands as empowering or as cold for someone having a hard time. Same app, same mechanics — only the words and the picture change.

How to run it. Show both screens to 5–8 people who’ve struggled with low mood. Counterbalance the order (half see A first, half see B first) and don’t say which one we made or prefer. Capture a one-word first reaction, then ask the questions below. Provide crisis resources; don’t recruit anyone in acute crisis.

The probe

  • First reaction, one word?
  • Does this feel like it’s on your side?
  • Does “being shaped / a stone” feel hopeful or harsh? (Variant A)
  • Would you want to open this tomorrow?
  • Which of the two would you choose — and why?

What the result means

Decision rule

Pass · build the stone

Stone framing is net-positive and not read as objectifying by most. “Unfinished, not broken” lands as hopeful. Proceed to the stone visual system.

Soften · retest once

Mixed — hopeful for some, cold for others. Keep the stone, lean on “not broken,” ease off “chiselled / struck,” run one more round.

Pivot · change the metaphor

Cold or objectifying for most. Swap the central metaphor before building. The mechanics in the spec are metaphor-independent; a warmer nurture image is the fallback.

Both mockups deliberately share one visual system — identical chrome, palette, buttons, and the same privacy line — so the test measures the framing, not which screen a designer made prettier. No mortality imagery appears in either, per the Phase 1 safety gate.
